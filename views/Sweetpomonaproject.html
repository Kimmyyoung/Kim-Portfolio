UI Design & Front-end Development

SWEETPOMONA Company Website

Timeline April 2022 - June 2022
Role UI Designer, Web Developer
Tools HTML5, CSS, Wordpress, WP Plugin Figma, Adobe Illustrator CS6
Overview

Sweetpomona is an trading company that sells coffee syrup and powder etc. Started in 2018, as cafe industry gets bigger and bigget, they aim promoting their products and service. Working as the only developer and designer in the team, the main focus was saving the time and maximized effection of main goal. I developed the ‘SWEETPOMONA’ website and designed graphic features for the website.

01

Plan

My client require to make website like E-catalogue with tight budget. Based on the client’s inquiry, I first composed with wireframe and brand features to estabilished brand identity. The breand features process took about a full-month. Composed with wireframe for website. Designed Brand Features.

02

Design

Once the idea made out of the conceptual stage and feedback has been finalized, I created design guidelines of the interface. Then I started to build a high fidelity design.

03

Implement

After finishing all of design tasks, I decided to use Wordpress to build website. Based on my client’s budget, I only use free WP plugin. In order to show the item list with pictures, I used a Woocommerce Plugin that is e-commerce plugin without price. (PS, selling the product was not a purpose for this website.) If there was some problem to design the site, I used “Theme Editor” for coding or login to webserver to modify style.css.
