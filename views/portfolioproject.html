Role UI Designer, Web Developer
Tools AWS, HTML5, CSS, jQuery, JavaScript, Figma, Adobe Illustrator CS6
Project Overview

This project was to build a portfolio web site that the user can see my works and projects. The website was built using Javascript, HTML5, CSS and jQuery.

01

Plan

First of all, I spent some time researching peopleâ€™s portfolios. I recognize how I would implement my portfolio by trying out some popular websites and what components would be included. Based on what I learned, I started to make a plan for my portfolio. In addition, I looked up all of my projects and decided what projects would be put in my portfolio.

02

Design

I created wireframes and mockups using Balsamiq and Figma to figure out how I would position the elements and style them. In this process, I struggled to choose the color for the website. Since I was making a website that only has a few pages, I decided to use a strong color that can give an impression to the user. This process helps me to get a sense of how I should design for users.

03

Implement

After finishing all of design tasks, I needed to decide what languages and plugin would be used. For the portfolio, I realized that I need to show and put all of my skills I can use. Tech Stack

  • Design Tools : Figma, Adobe Photoshop CS6, Adobe Illustrator CS6
  • Web languages : HTML5, CSS, Vanilla JS, jQuery
  • Plugin : AOS Plugin, Bootstrap
  • Server & Deploy: AWS, Node.js
