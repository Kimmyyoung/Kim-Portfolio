ECOPACCANADA E-commerce Website

Timeline August 2020 -
January 2021
Role UI Designer,
Web Developer
Tools HTML5, CSS, Wordpress, Figma, Adobe Illustrator CS6
Project Overview

GEN Trading (DBA) ECOPACCANADA is a food packaging distributor where located in Burnaby, BC, Canada. Through ECOPACCANADA, the user can enrich their best deal of business solutions. This project was to make the company website. The main purpose was to promote business and to introduce company’s items. Working as the only designer and developer in the team, I designed and prototyped the first version of a website.

01

Plan

First of all, I studied what the company is doing and selling. Based on study, I created a new design concept revamping the prototype that was made from previous empolyee, and built the website to accomplish company’s needs. 1. Redesigned website theme and layout product list. 2. Created all desgin features for products.

02

Design

I composed website with Figma according to the CEO’s inquiry. The main objective of this was to show the item description and sell the products successfully. Based on prototype I composed previous chpater, the color and theme was generated.

03

Implement

After finishing all of design tasks, I decided to use Wordpress to build website. Based on my client’s budget, I only use free WP plugin. In order to show the item list with pictures, I used a Woocommerce Plugin that is e-commerce plugin without price. (PS, selling the product was not a purpose for this website.) If there was some problem to design the site, I used “Theme Editor” in Wordpress for coding or login to webserver through Filezilla to modify style.css.
