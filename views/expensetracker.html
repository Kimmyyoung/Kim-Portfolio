Expense Tracker Project

Timeline December 2022
Role Front-end Developer, UI Designer
Tools Development : ReactJS, Styled-component | Design : Figma
Project Overview

The expense tracker project is a service for tracking income and expense. This project was built using React JS and Styled-component.

01

Wireframe

When I studied and used the ReactJS for my project, I had a struggle to make a component tree. Sometimes, I should recode the components because I did not properly build for the first time.

I started this project to practice making components tree. And I used a styled-component which is familiar to me for style sheet.

02

Components Tree

PocketContainer

  • PocketStatus : Summary of Transaction
  • PocketHistory : Details of Transaction
  • PocketChart : Graphic User Interface for transaction amount

NewItemContainer

  • Category : Category of Transaction
  • NewItemDate : Input (type: date)
  • Amount : Input (type: text)
  • Button : Button (type: button)
