Full-stack Development

Study Blog Project

Timeline March 2021
Role UI Designer, Full-stack Developer
Tools HTML5, CSS, JavaScript, Bootstrap, jQuery, MongoDB, NodeJS, AWS Ubuntu (EC2)
Project Overview

This project was to build a simple blog web site where the user can create and delete notes about what they studied. The website was built using Javascript, MongoDB, HTML5, CSS and Material UI.

01

Plan

Towards my previous projects, I realized that lack my data skills. For my skill-up, I decided to make a website using MongoDB. I spent some time thinking about what I would make and how I would divide sections to show data. Through my blog project, I wanted to make what I will really use in my real life to share my lifestyle and my study etc.

02

Design

I created a wireframe and design with Figma to figure out how I would position the elements and style them. I wanted to design with a simple theme and focus on how I would show the information and purpose. Also, I decided what languages I would use for this project.

Tech Stack

Front-end : HTML5 | CSS | Javascript
Back-end : Node.js
Database : MongDB

03

Implement

After finishing up HTML markup and styling using CSS, I added some animations using the Bootstrap library to make a great layout. In order to save and get data, I used MongoDB and Node.js. Through this process, I learned how to use non-SQL.

04

Deploy

To deploy this project, we decide to use Amazon Web Service. Since this project is a personal project, I could use the free tier for EC2. I made a cloud server with Ubuntu which is familiar to me. and deployed the project from Git where I controlled the version of the project. I needed to redirect the domain (port) because this project was developed using node.js. So I used 'redir' command in my cloud server. With this experience, I solved the error of deployment, I learned that I can deploy any project with AWS.
